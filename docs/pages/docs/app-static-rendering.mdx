# Static Rendering

<Callout type="warning" emoji="⚠️">
  This feature doesn't work properly yet (still generates Functions instead of static HTML) despite that `next build` properly marks the pages as SSG.
</Callout>

Export `getStaticParams` from `createI18nServer`:

```ts {3}
// locales/server.ts
export const {
  getStaticParams,
  ...
} = createI18nServer({
  ...
})
```

Then, export a new `generateStaticParams` function from your root layout. Next.js requires it to be a function, so you can't just export the `getStaticParams` function directly:

```ts {2,4-6}
// app/[locale]/layout.tsx
import { getStaticParams } from '../../locales/server'
 
export function generateStaticParams() {
  return getStaticParams()
}
```
